## Step 2: Details

**Project Details**
> **Xeus** is a browser-native IDE for building, testing, and deploying Solana AI agents — with zero local setup. Developers can open a browser tab, write TypeScript agent logic (or describe it in English), and watch their agent execute real Devnet transactions in an interactive chat panel.
> 
> **The Problem**: Building a Solana AI agent today requires installing Node.js, npm, TypeScript, cloning a repository, getting OpenAI and Helius API keys, configuring `.env` files, and running a local server. This 45-minute setup barrier kills developer momentum before a single transaction runs. While tools like Solana Agent Kit (SAK) solve the code library problem, there is no browser-native playground to run, test, and share agents instantly.
> 
> **The Solution**: Xeus provides a pre-configured Monaco editor workspace with live TypeScript compilation, integrated with a Devnet wallet API and built-in on-chain tools (getBalance, transferSOL, swapTokens, mintNFT). Developers can edit agent behavior and test it immediately in a live chat interface, making it the perfect "test track" for the SendAI ecosystem.
